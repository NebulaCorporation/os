5 reasons why NebulaOS is better than Stock Windows 10.

LightweightSystem

Thanks to the disabled unnecessary services and removed bloatwares, NebulaOS needs only a few processes to work, making the system as light as possible.

memory

Low CPU / RAM / DiskUsage

The difference is noticeable, NebulaOS is just using 500MB of RAM in idle.
With NebulaOS, you are now able to dedicate more RAM to your games, especially on low memory systems.

Fully workingout of the box

NebulaOS uses automated scripts during and after the OS' installation to minimize the actions that the user must perform.
